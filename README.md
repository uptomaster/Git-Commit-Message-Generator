# Git Commit Message Generator

브라우저에서 입력만으로
`git commit -m "..."` 명령어를 자동 생성하는 초소형 웹 도구입니다.

---

## Features

* `git commit -m ""` 자동 생성
* 브랜치 이름 선택 반영
* 작업 상태(작업중 / 완료) 포함
* 작성자 이름 포함
* 원클릭 복사 + 토스트 알림
* 순수 HTML / CSS / JS (서버 없음)

---

## Example

```bash
git commit -m "[lnh] 로그인 API 완료 (이남혁)"
```

---

## Usage

1. `index.html` 실행
2. 정보 입력
3. 생성된 커밋 명령 복사 → 터미널 실행
